## 一、逢事易询是什么

**逢事易询**是一款 AI 命理咨询助手。你只需要像和朋友聊天一样说出心中的困惑，它就能用传统命理智慧帮你分析、解读、找方向。

**一句话**：用人话聊命理，让古老智慧为你所用。

---

## 二、能做什么

### 2.1 六种算法，同一个对话就能切换

逢事易询内置了六套完整的传统命理算法。你不需要懂专业术语，只需要自然地说出想问什么，它就会选最合适的角度来解答。想换个角度？在对话里直接说就行，它会在同一次聊天里切换不同的算法来交叉验证，给出的结论更立体。

| 算法 | 适合问什么 |
|------|-----------|
| **紫微斗数** | 人生全局、事业走向、财运起伏、感情姻缘 |
| **六爻** | 具体某件事的走向、短期动向、决策参考 |
| **八字** | 命格特点、五行喜忌、婚姻匹配、大运走势 |
| **四柱命卦** | 本命卦、后天需要注意的方向 |
| **小六壬** | 速断吉凶、寻找失物、出行参考 |
| **取名** | 宝宝取名、改名建议（五行补益 + 康熙字典取字） |

### 2.2 不止看本命，更要看"什么时候"

知道了命盘的全貌还不够——运势是会随着时间流动的。逢事易询支持从年到时的四层流转，帮你找到属于自己的时间窗口。

#### 紫微斗数：大限 → 流年 → 流月 → 流日 → 流时

你的命盘十二宫每一宫都有一个"大限"，代表十年一换的人生阶段。每个大限期间，不同宫位被激活，人生重心自然不同。在对话里，AI 会自动结合你当前所处的大限来解读，让你知道现阶段最该关注什么。

在大限的基础上，你可以一层层往细了看：

| 粒度 | 能看什么 | 举例 |
|------|---------|------|
| **流年** | 今年整年的运势重点、关键月份 | "今年事业上有机会升职吗" |
| **流月** | 某个月具体会发生什么 | "下个月适合跳槽吗" |
| **流日** | 某一天的能量状态 | "后天去谈合作运气怎么样" |
| **流时** | 某个时辰的吉凶 | "签约选上午还是下午" |

每一层都对应不同的四化飞星和宫位流转，从年的大势到时辰的精准节点，逐层细化。聊着聊着你自然就能找到最合适的时机。

#### 八字：十年一步大运

八字的大运体系以十年为一步，每步大运都有明确的干支组合和十神配置，决定了你这十年的整体基调。大运一变，人生方向往往随之转折。

AI 在解读时会自动帮你定位当前处于哪步大运、离换运还有几年，结合流年干支与命局、大运的交互，告诉你当前阶段的重点和拐点。

#### 四柱命卦：流年 → 流月 → 流日

命卦体系的流转用卦象表达——每个时间周期对应一个卦，每个卦对应一条爻辞。从卦的变化中，可以看到事情的吉凶方向和注意事项。支持年、月、日三层流转。

### 2.3 不是冰冷的结果页，而是"聊"出来的答案

传统的算命工具只会丢给你一堆术语和表格。逢事易询不一样——它以对话的方式和你互动：

- **像和真人聊天一样**：你不用选菜单、不用填复杂表单，直接把想问的说出来就好
- **信息不够它会主动问**：比如你问事业，它会追问时间范围、具体方向，就像一个有经验的命理师在和你交流
- **一次对话追问到底**：排盘、解读、追问、换个角度再看，全在同一个聊天窗口里完成
- **每轮只讲透一个方向**：不铺开一堆你不关心的内容，围绕当前问题深入解答，信息刚好够消化
- **想换话题随时换**：聊完事业还想看感情，直接在对话里说，它会自动切换算法和解读角度

### 2.4 感情的事，聊得清楚

感情问题是最常问的，也是最需要看全面的。

**看正缘**：AI 会结合八字的夫妻星、紫微的夫妻宫来看。你的正缘大概是什么时候来、对方可能具备什么特征——这些在命盘里都有线索。

**看合婚**：有对象了想知道合不合适？你和对方的八字放在一起看——日柱是否相合、五行是否互补、夫妻宫是否相配，AI 会逐项分析。缺了对方的信息也没关系，可以先看自己这边的倾向和需要注意的地方。

**看桃花**：流年桃花星动没动、在哪一宫、是好桃花还是烂桃花——这些在紫微流运里一目了然。

### 2.5 懂你的出生地——真太阳时

你可能不知道，出生时间用"北京时间"其实不够准。比如你出生在成都，实际的太阳时比北京时间晚大约一小时——这对排盘结果有直接影响。

逢事易询在填写生辰时会让你选择出生城市（省 → 市 → 区县三级联动），自动换算当地的真太阳时后再排盘。如果你不确定出生城市，默认会用北京。

## 四、常见问题

**Q：我不会算命，能用吗？**
A：完全不需要任何基础。你就用大白话说出想问的，AI 会自动选择合适的算法来排盘和解读。你只需要聊。

**Q：算的准吗？**
A：逢事易询使用的是传统命理的正统算法（紫微斗数、六爻、八字等），排盘过程由算法精确完成。解读部分由 AI 基于命盘数据生成——它提供的是一个有依据的方向参考，最终的选择和判断还是在你手中。

**Q：需要付费吗？**
A：每天可以免费开启一个新对话，足够日常使用。如果想聊更多，可以通过分享或看广告解锁，也可以 ¥1 付费跳过。

**Q：我的出生时间不记得精确到几点怎么办？**
A：尽量精确就行。如果能记住大概的时间段（比如上午/下午/傍晚），填一个大概时间，AI 也能正常排盘。出生城市尽量选准确，会影响真太阳时换算。

**Q：想帮家人朋友问，可以吗？**
A：可以。你可以在「关注的人」里添加家人朋友的生辰信息，然后在对话中提及到对应的人来问。不用反复填写，一次录入，随时切换。
